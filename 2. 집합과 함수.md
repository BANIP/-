# 집합과 함수

이 장에서는 이산수학을 이해하기 위한 기본 개념인 집합과 함수를 다룬다. 집합과 함수 개념은 전산학에서 알고리즘의 복잡도 계산에 사용될 수 있고 여러 증명법의 기본을 제공한다.

## 집합(set)
### 정의
이산 객체들과 그들간의 관계를 표현하는 가장 기본적인 개념이다. 집합은 **서로 다른(distinct) 객체들의 모임** 이다. 집합의 성질은 아래와 같다.
* 집합은 서로 다른 객체의 모임이기 때문에 **동일한 원소를 두 개 이상 포함 할 수 없다**.
* 집합의 객체는 **순서개념이 없다**.

집합의 정의에 의하면 집합은 서로 다른 객체의 모음이기 때문에 `{a,a,b,c}`와 같은 사용법은 `{a,b,c}`의 잘못된 표현이다. 또한 순서개념이 없기 때문에 `{a,b,c}`와 `{a,c,b}`는 같은 집합을 나타냄을 알 수 있다.

또한 집합안에 다른 집합을 포함할 수 있다. 한 대학에서 1학년으로 이루어지는 그룹은 한 집합이 되고, 1학년중 컴퓨터를 전공하는 학생의 그룹도 또 하나의 집합이 된다.

### 기본적인 표현법

집합의 표시는 `S = {a,b,c}`의 형태로 나타낼 수 있다. 이 집합을 세세한 부분이 의미하는 바는 아래와 같다.

* **S** : 집합의 이름
* **a, b, c** : 집합의 원소(element)

또한 집합에 소속되어 있는 원소를 밝히는 용도로 `a ∈ S`와 같이 사용하는데, 이는 원소 a가 집합 S의 원소임을 나타내며 `d ∉ S`는 d가 집합 S의 원소가 아님을 나타낸다.

### 집합의 다양한 표현법

집합은 두가지 표현법이 있다. 이 두가지 방법은 표현하려는 값은 같아도 표현하는 방법이 다르다. 각각의 표현법을 이용해 집합을 표현하면 아래와 같아진다.

* **조건 나열법** : `S = {1,3,5,7,9}`
* **조건 제시법** : `S = {x|x는 10보다 작은 홀수}`

### 집합의 성질을 이용한 예
집합의 정의에 의하면 그 원소가 없는 경우를 제외하지 않는다. 따라서 빈 집합(empty set)이 존재하며 그 기호로는 {}이나 ∮와 같이 나타낸다. 정의에서 말한바와 같이 집합은 다른 집합을 원소로 가질 수 있다. 예륻 들어 다음과 같은 집합이 주어졌다고 하자.
* A₁ = `{a,b}`
* A₂ = `{{a,b}}`
* A₃ = `{{{a,b}}}`
이 집합에서는 다음과 같은 식이 성립함을 알 수 있다.
* `a ∈ A₁`
* `a ∉ A₂`
* `a ∉ A₃`
* `A₁ ∈ A₂`
* `A₁ ∉ A₃`
* `A₂ ∉ A₃`

### 부분집합
또한 두 집합간의 관계에 있어서 부분집합은 P의 모든 원소가 Q의 원소가 되면 **P는 Q의 부분집합이다** 라고 할 수 있다. 기호로는 `P ⊆ Q`로 표현된다.

> 예제) 술어 `P(x,y)`가 "x는 y의 부분집합이다."라는 문장일 때, `{a,b}`인 x와 `{a,b,c,d}`인 y, `{a,c,d}`인 z에 대하여 `P(x,y)`와 `P(x,z)`의 진위를 판별하여라.
> ***
> 첫번째 논리식은 x의 원소인 a,b가 y에 모두 포함되어있으므로 참이며 z와의 관계는 b가 그렇지 않으므로 거짓이다.

부분집합의 정의에 의하면 다음과 같은 성질이 존재한다.

1. `∀P`에 대하여 P는 P의 부분집합이다.
2. `∮`은 모든 집합의 부분집합이다. 다만 `∮`이 모든 집합의 원소가 되는 것은 아니다.
3. 집합 `{∮}`는 집합 `{{∮}}`의 부분집합이 아니지만 집합 `{∮}`는 `{{∮}}`의 원소이다.

다음으로 다음과 같은 두 집합에 대해 생각 해 보자.

> * `P = {x|x는 10보다 작거나 같으며 0보다 큰 짝수의 집합}`
> * `Q = {x|x = y + z, y ∈ {1,3,5}, z ∈ {1,3,5}}`
두 집합은 같은 집합을 의미한다. 두 집합이 같다는 말은 그 각각이 같은 객체를 포함하고 있다는 의미이다. 두 집합 P와 Q가 서로의 부분집합 관계에 있으면, 그 집합은 같다와 동일한 의미이다.

## 집합의 조합
이 절에서는 둘 이상의 집합이 새로운 집합을 생성하는 방법에 대해 다룬다. 예를 들어, 집합 A와 집합 B를 다음과 같이 정의하기로 하자.

* `A = {x|x는 한 대학에서 이산 수학 과목을 듣는 학생}`
* `B = {x|x는 한 대학에서 윈도우 프로그래밍 과목을 듣는 학생}`

이 경우 우리는 이 두 집합간의 여러 관계에 대해 생각할 수 있다. 만일 학교의 어떤 뉴스가 이산수학과 윈도우 프로그래밍 수업 시간에 발표되었다고 하자. 그 뉴스를 들은 학생의 집합은 이산수학을 듣거나, 윈도우 프로그래밍 수업을 듣거나, 혹은 두 과목 모두를 듣는 사람중 하나일 것이다. 또, 이산수학과 윈도우 프로그래밍이 같은 시간에 시험을 치루기로 예정되어 있으면 시험이 겹친 학생의 집합은 이산 수학과 윈도우 프로그래밍을 동시에 듣는 학생일 것이다. 

이렇게 집합간의 관계로 새로운 집합을 만들어내기 위해 여러 집합의 조합이 존재한다.

### 합집합(union)
A ∪ B로 표현되는 집합 A와 집합 B의 합집합은, 그 원소로 **A 또는 B, 혹은 A와 B의 원소로 이루어진 집합** 을 의미한다.

> 예제) 다음 두 집합의 합집합을 구하여라. 
   > 1. `{a,b} ∪ {c,d}`
   > 1. `{a,b} ∪ {a,c}`
   > 1. `{a,b} ∪ ∮`
   > 1. `{a,b} ∪ {{a,b}}`
   >***
   > 1. `{a,b,c,d}`
   > 1. `{a,b,c}`
   > 1. `{a,b}`
   > 1. `{a,b,{a,b}}`

### 교집합(intersection)
`A ∩ B`라고 표현되는 집합 A와 집합 B의 교집합은 **A의 원소이면서 동시에 B의 원소로 이루어진 집합** 을 의미한다.
> 예제) 다음 두 집합의 교집합을 구하여라
> 1. `{ a, b } ∩ {a,c}`
> 1. `{ a, b } ∩ {c,d}`
> 1. `{ a, b } ∩ ∮`
>***
> 1. `{a}`
> 1. `∮`
> 1. `∮`

### 조합의 성질
#### 교환법칙
정의에 의하면 다음의 합집합과 교집합은 교환법칙이 성립한다. 즉,
* `A ∪ B = B ∪ A`
* `A ∩ B = B ∩ A`

는 참이 된다는 의미이다.

#### 분배법칙
분배법칙 또한 성립한다. `R ∩ ( P ∪ Q) = (R ∩ P) ∪ (R ∩ Q)`은 참이된다.

이에 대한 증명은 다음과 같다.
* `R ∩ (P ∪ Q) ⊆ ( R ∩ P ) ∪ (R ∪ Q)`임을 증명한다.
*  `x ∈ R ∩ ( P ∪ Q )`라면, 교집합의 정의에 의해 `x ∈ R`이고, `x ∈ P ∪ Q`이어야 한다. 
* 이때 두번째 식에서,
    * `x ∈ P`이면 `x ∈ R ∪ P`에 속한다.
    * `x ∈ Q`이면  `X ∈ R ∪ Q`가 된다. 
* 따라서 `x ∈ R ∩ ( P ∪ Q )`가 된다. 
* 결국 `R ∩ (P ∪ Q) ⊆ ( R ∩ P ) ∪ (R ∪ Q)`임을 알 수 있다.

>예제) P,Q,R의 집합을 아래와 같다고 정의하자.
>* `P = {x|x는 한 대학에서 이산 수학 과목을 듣는 학생}`
>* `Q = {x|x는 한 대학에서 윈도우 프로그래밍 과목을 듣는 >학생}`
>* `R = {x|x는 혈액형이 AB형인 학생}`
>
>현재 교내에는 이산 수업 과목과 윈도우 프로그래밍 과목을 듣는 학생이 수업을 듣고 있다. 헌혈 대상차를 찾기 위해 방송으로 혈액형이 AB형인 학생을 찾는다고 할 때 이에 해당되는 학생의 집합을 두개 구하여라.
>***
>* **<center>(P ∪ Q) ∩ R</center>**
>* **<center>(P ∩ R) ∪ (Q ∩ R) </center>**

### 차집합
차집합은 `A - B`로 표현된다. 차집합은 B의 원소에 속하지 않는 A의 원소로 이루어진 집합이다.

> 예제) 아래 집합 조합식을 간략화하여라.
>
> 1. `{ a, b, c } - {a}`
> 1. `{ a, b, c } - {a, d}`
> 1. `{ a, b, c } - {d, e}`
> ***
> 1. **`{b,c}`**
> 1. **`{b,c}`**
> 1. **`{a,b,c}`**

#### 대칭차집합
두 집합 간의 대칭 차집합은 **A또는 B에 존재하는 원소이고, A와 B의 교집합에 포함되지 않는 원소** 를 의미한다. 즉 집합의 조합식으로 표현하면 `(A ∪ B) - (A ∩ B)`이며 기호로 표현하면 **`A △ B`**이 된다.

> 예제) 아래 집합 조합식을 간략화하여라.
>
> 1. `{a,b} △ {a,c}`
> 1. `{a,b} △ ∮`
> 1. `{a,b} △ {a,b}`
> ***
> 1. **`{b,c}`**
> 1. **`{a,b}`**
> 1. **`∮`**

> 예제) 어떤 학교의 학생이 이산수학을 받는 학점의 경우는 아해와 같다.
>* A학점 => 2번의 퀴즈에서 점수를 잘 받아야 함
>* B학점 => 1번의 퀴즈에서 점수를 잘 받아야 함
>* C학점 => 모든 퀴즈에서 점수를 못받아야 함 함
또한 집합 P와 Q의 정의는 아래와 같다.
>* P => 첫번째 퀴즈에서 점수를 잘받은 학생의 집합
>* Q => 두번째 퀴즈에서 점수를 잘받은 학생의 집합
>이 때, 모든 학점의 경우를 P와 Q의 조합식으로 표현하여라.
>***
>* **A학점 => P ∩ Q** 
>* **B학점 => P △ Q** 
>* **C학점 => S (P ∪ Q)** 

####멱집합(power set)
P(A)로 표현되는 멱집합은 **집합 A의 모든 부분집합을 그 원소로 가지는 집합** 을 의미한다.

즉, `P({a,b})`이면 이에 해당하는 집합은 `{∮,{a},{b},{a,b}}`에 해당하며 `P({ })`이면 `{{ }}`에 해당한다. 이 때, 멱집합의 성질에 의해 모든 집합 A에 대하여 `{ } ∈ P(A)`이면서 `{ } ⊆ P(A)`이다.

### 유한 집합(finite set)과 무한 집합(infinite set)
집합의 크기라고 하면 그 원소의 갯수로 정의되는것이 가장 합리적일 것이다. 예를들어 집합 `{a,b,c}`의 집합의 크기는 3이며 `{{a,b}}`의 집합의 크기는 1일 것이다. 이 장에서는 이러한 유한집합의 개념과 이와 반대되는 무한집합의 개념을 다루게 될 것이다. 무한집합의 경우는 그 크기를 명시할 수 없기 때문에 countable과 not countable이 이슈가 된다.

#### 계승자(Successor)
먼저 무한집합과 유한집합을 구분하기 위해 새로운 집합 구분자를 정의한다. A<sup>+</sup>로 표시되는 A의 계승자는 `A ∪ {A}`로 정의된다.

이게 무슨말이냐면 A가 `{a,b}`이면 `A`<sup>`+`</sup>은 `{a,b} ∪ {{a,b}}`이므로 `{a,b,{a,b}}`가 된다는 말이다.

그러면 다음과 같은 집합을 생각할 수 있다.
```
    0 = ∮
    1 = { ∮ }
    1 = { ∮, { ∮ } }
    1 = { ∮, { ∮ }, { ∮, { ∮ } } }
```
이때 계승자의 정의에 의해 1 = 0<sup>+</sup>, 2 = 1<sup>+</sup>, 3 = 2<sup>+</sup>이 됨을 알 수 있다. 따라서 다음과 같이 집합 N을 정의한다.
1. 집합 N은 집합 0을 포함한다.
1. 만일 집합 n이 집합 N의 한 원소라면 집합 n
<sup>+</sup>도 집합 N의 원소이다.
1. 집합 N은 다른 집합을 포함하지 않는다.

#### 1:1대응관계
정의에 의하면 집합 `N={0,1,2,3,…}`의 형태로 명확한 무한집합이 된다. 이때 집합 N은 명확하게 셀 수 있는 형태이다. 여기서 셀 수 있는것을 보이기 위해 두 집합 1:1 대응을 정의한다.

이때 정의 내용은 집합 P, Q가 있으면, **P의 모든 원소가 Q의 서로 다른 원소가 짝을 지을 수 있으면** 이 때 집합 P의 원소와 집합 Q의 원소간에 **1:1 대응 관계** 가 있다고 한다.

> 예제) 집합 {a,b}의 각 원소와 집합 {c,d}의 각 원소는 서로 1:1 대응 관계에 있다. 그 이유를 설명하라.
>***
>**<center>두 집합의 원소는 갯수가 같기 때문에 서로 짝을 지을 수 있기 때문이다 예를 들어 a는 c, b는 d와 같이 짝을 짓는것이 가능하다.</center>**

#### 유한집합
어떤 집합의 원소와 `n ∈ N` 인 집합의 n의 원소간에 1:1 대응 관계에 있으면 그 집합은 **유한 집합(cardinality)** 이고 그 크기는 n이다.

집합에 대한 크기는 집합 내의 원소의 갯수로 알 수 있으며, 예를 들어 집합 `{a,b,c}, {a,∮,d}, {∮, {∮}, {∮, {∮}}`는 모두 크기가 3임을 알 수 있다.

#### 무한집합
##### 셀 수 있는 무한집합
다음으로 어떤 집합이 셀 수 있는 무한집합이라고 하면, **어떤 집합의 원소와 집합 N의 원소간에 1:1 대응 관계에 있다** 는 것을 의미한다.

예를들어 집합 `{0,1,2,3,…}`은 명확히 셀 수 있는 무한집합이고 집합 `{0,2,4,6,…}`또한 셀 수 있는 무한집합이다. 집합 `{0,2,4,6}`의 경우에는 원소 `i=0,1,2,…`에 대해서 2i로 대응 될 수 있다.

이러한 무한집합의 성질은
* 두개의 셀 수 있는 무한집합의 합집합 역시 셀 수 있는 무한집합이다.
* 셀 수 있는 무한집합이 유한개 존재할 때 그들의 합집합 역시 셀 수 있는 무한집합이다.
* 셀 수 있는 무한집합이 셀 수 있게 무한개 존재할 때 그들의 합집합도 셀 수 있는 무한집합이다.

들이 존재한다.

##### 셀 수 없는 무한집합
이 절에는 셀 수 없는 무한집합이 존재하는것을 보인다. 이러한 예중 하나는 0과 1사이의 실수의 집합이라고 볼 수 있다. 이러한 명제에 대한 증명은 아래와 같이 할 수 있다.

먼저 0과 1사이의 실수는 집합 R이 셀 수 있는 무한 집합이라고 가정할 때, 집합 R이 셀 수 있는 무한집합이라면 정의에 의하여 R집합의 원소와 N 집합의 원소가 1:1 대응관계까 성립한다. 그러면 우리는 ㅈ다음과 같이 집합 R이 원소를 나열할 수 있다.


* `0.a₁₁a₁₂a₁₃a₁₄…` 
* `0.a₂₁a₂₂a₂₃a₂₄…` 
* `0.a₃₁a₃₂a₃₃a₃₄…`
* …………
* 0.a<sub>i1</sub>a<sub>i2</sub>a<sub>i3</sub>a<sub>i4</sub>…

여기서는 a<sub>ij</sub>는 집합의 원소를 나열했을 때 i번째 수로 j번째의 0-9사이의 숫자를 의미한다. 그러면 다음의 수를 생각 해 보자.
**<center>`P = o.b₁b₂b₃b₄…`</center>**
* b는 a<sub>ij</sub> = 9 일때  1
* b는 0 < a<sub>ij</sub> < 9일때  9 - a<sub>ij</sub>

이러면 P는 0과 1 사이의 실수이지만 위에서 리스트할 집합 R의 어느 한 원소와도 같지 않다. 왜냐하면 P는 R의 원소중에 a<sub>ij</sub> 항목이 반드시 다르기 때문이다. 따라서 0과 1의 사시의 셀 수 있는 무한 집합이라는 가정이 옳지 않음을 알 수 있다. 따라서 0과 1의 사이의 실수는 셀 수 없는 무한집합이다.

## 수학적 귀납법
이 절에서는 수학에서 많이 쓰이는 증명법인 수학적 귀납법을 다룬다. 자연수 n에 연관된 문장을 증명하기 위하여 우리는 다음의 기법으로 증명할 수 있다.
1. `n = n₀`일 때 그 문장이 참임을 보임
2. 이 문장이 `n = r( r ≥ n₀)`일 때 참을 가정하고 `n = k + 1`일때 참을 보임.

이때 우리는 그 문장이 `n ≥ n₀`인 모든 자연수에 대해 참이라고 결론지을 수 있다.


> 예제) 다음을 증명하시오.
**<center>![1^2 + 2^2 + ... + n^2 = \frac{n(n+1)(2n+1)}{6} (n\geq1)](https://latex.codecogs.com/svg.latex?\inline&space;\dpi{100}&space;\bg_white&space;\LARGE&space;1^2&space;&plus;&space;2^2&space;&plus;&space;...&space;&plus;&space;n^2&space;=&space;\frac{n(n&plus;1)(2n&plus;1)}{6}&space;(n\geq1))</center>**
> ***
> 1. `n = 1`인 경우, ![](https://latex.codecogs.com/svg.latex?\dpi{100}&space;\bg_white&space;\large&space;1^2&space;=&space;\frac{1&space;\cdot&space;2&space;\cdot&space;3}{6})이므로 성립함.
> 2. n = k일 때, 성립합을 가정하면 ![1^2 + 2^2 + ... + k^2 = \frac{k(k+1)(2k+1)}{6}](https://latex.codecogs.com/svg.latex?\inline&space;\dpi{100}&space;\bg_white&space;\large&space;1^2&space;&plus;&space;2^2&space;&plus;&space;...&space;&plus;&space;k^2&space;=&space;\frac{k(k&plus;1)(2k&plus;1)}{6})이 성립한다. 
    >
    >   이때 이 식의 양변에 `(k + 1)²`를 더하면
    > ![1^2 + 2^2 + ... + k^2 + (k+1)^2 = \frac{k(k+1)(2k+1)}{6} + (k+1)^2](https://latex.codecogs.com/svg.latex?\inline&space;\dpi{100}&space;\bg_white&space;\LARGE&space;1^2&space;&plus;&space;2^2&space;&plus;&space;...&space;&plus;&space;k^2&space;&plus;&space;(k&plus;1)^2&space;=&space;\frac{k(k&plus;1)(2k&plus;1)}{6}&space;&plus;&space;(k&plus;1)^2)
>   ![= \frac{(k+1)(k(2k+1) + 6(k+1))}{6}](https://latex.codecogs.com/svg.latex?\inline&space;\dpi{100}&space;\bg_white&space;\LARGE&space;=\frac{(k&plus;1)(k(2k&plus;1)&space;&plus;&space;6(k&plus;1))}{6})
    ![= \frac{(k+1)(2k + 3)(k + 2)}{6}](https://latex.codecogs.com/svg.latex?\inline&space;\dpi{100}&space;\bg_white&space;\LARGE&space;=\frac{(k&plus;1)(2k&space;&plus;&space;3)(k&space;&plus;&space;2)}{6})
    ![= \frac{(k+1)(k+2)(2(k+1) + 1)}{6}](https://latex.codecogs.com/svg.latex?\inline&space;\LARGE&space;=\frac{(k&plus;1)(k&plus;2)(2(k&plus;1)&space;&plus;&space;1)}{6})
    >
    > 따라서 주어진 식은 `n = k + 1`인 경우에도 성립한다.
    >
    > 그러므로 해당 식은 `n ≥ 1`에 대해서 성립한다.

> 예제) 한 나라의 왕이 그들 신하가 얼마나 영리한지 알기 위하여 다음과 같이 신하에게 말하였다고 한다.
>
>     내가 너희들중 일부에 하얀색 모자를 씌우고 나머지에게는 검정색 모자를 씌울 것이다. 너희들은 서로 말할 수 없고 자신의 모자 색은 볼 수 없지만 남들이 쓴 모자 색은 볼 수 있다. 내가 한시간에 한번씩 방에 들어 올 텐데 너희들 중 자신이 하얀색 모자를 쓰고 있다는 사실을 아는 순간에 말하여야 한다.
>
> 이 문제는 하얀색 모자를 쓴 신하가 n명이면 그들은 모두 n시간 안에 그들 자신이 하얀색 모자를 쓰고 있다는 사실이 증명되었다. 이것을 수학적 귀납법으로 증명하여라.
> ***
> * `n = 1`일 때, 하얀색 모자를 쓴 신하가 1명이면 그 신하는 왕이 첫번째 들어왔을 때 자신이 하얀색 모자를 썼다는 사실을 알 수 있다. 왜냐하면 그 신하는 다른 신하가 모두 검은 모자를 쎴는데 한명 이상은 하얀색 모자를 씌운다고 했으므로 자신이 하얀색 모자를 썼다는 사실을 알 수 있다.
> * `n = k`일 때, 즉 하얀색 모자를 쓴 신하가 k명일 때 그들은 k시간 안에 그들이 하얀색 모자를 썼다는 사실을 안다고 가정하자. 만일 하얀색 모자를 쓴 사람을 알 수 있는데 k시간
째 내에 아무도 안나간다는 사실에서 자신이 하얀 모자를 쓰고 있다는 사실을 알 수 있다. 왜냐하면 자신이 검정 모자를 쓰고 있고 하얀 모자를 쓴 사람이 k명이면 가정에 의해 k시간에 하얀 모자를 쓴 사람은 자신이 하얀모자를 쓴 사실을 알았을 것이기 때문이다. 그런데 아무도 안나간 사실에서 하얀 모자를 썼다는 사실을 알 수 있고 `k+1`시간에 자기가 하얀 모자를 쓰고 있다는 사실을 말할 수 있다.

> 예제) 조각맞추기 장난감이 있다고 하자. 이 장난감은 다음과 같은 특성을 지닌다.
> 
> * 하나 이생의 조각이 아귀가 맞으면 큰 하나의 조각으로 맞추어진다. 
> * 블록을 하나의 조각이거나 두개 이상의 조각이 아귀가 맞아서 이루어진 큰 조각이라고 가정한다.
> 
> 조각 맞추기 게임은 작은 블록들을 모아 결국 하나의 큰 블록을 만드는 게임이다. 아귀가 맞는 두개의 블록을 하나의 블록으로 맞추는 과정을 하나의 스텝이라고 하자. 수학적 귀납법을 이용해서 `n`개의 조각으로 이루어진 조각 맞추기 게임이 끝나기 위해서 `n-1`의 스텝이 항상 필요하다는것을 증명하여라.
> ***
> `n = 1`일 때, 조각이 하나일 경우 스텝이 필요없이 게임을 종료하므로 성립한다.
>
> `n = k`일 때, 즉 `(1 ≤ n ≤ k) n - 1` 스텝이 필요하다고 가정하자. 게임에 필요한 모든 조각이 `k + 1`로 이뤄져 있다고 가정하면 마지막 스텝의 바로 이전에 두개의 블록이 존재 할 것이다. 하나의 블록이 n₁조각으로 이루어지고 다른 하나가 n₂조각으로 이루어진다고 가정하자. 여기서 `n₁ + n₂ = k + 1`이 된다. 가정에 의하면 마지막 남은 두개의 블록은 n₁과 n₁스텝으로 만들어졌을 것이다. 따라서 마지막 스텝을 포함하여 게임을 종료시키기 위해 (n₁ - 1) + (n₂ - 1) + 1 = k번의 스텝이 필요함을 알 수 있다. 즉 n = k + 1일 때, k번의 스텝이 필요함을 알 수 있다.

## 집합 연산의 원소 갯수
이 절에는 유한집합에 대해 원소의 갯수에 대한 내용을 다룬다. `|P|`라는 기호를 집합 P의 원소의 갯수로 정의하면 다음과 같은 사실이 알려져 있다. 증명은 생략한다.
* `|P ∪ Q| ≤ |P| + |Q|`
* `|P ∩ Q| ≤ min(|P|,|Q|)`
* `|P △ Q| = |P| + |Q| - 2|P ∩ Q|`
* `|P - Q| ≥ |P| - |Q|`
* `|P ∪ Q| = |P| + |Q| - |P ∩ Q|`
* `|P ∪ Q ∪ R| = |P| + |Q| + |R| - (|P ∩ Q| + |Q ∩ R| + |R ∩ P|) - |P ∩ Q ∩ R|`

> 예제) 총 학생 200명을 보유한 어떤 학교에서는 다음과 같은 학생들이 이런 과목을 듣는다고 한다.
>
> *  50명은 이산수학 과목을 듣는다.
> * 140명은 윈도우 프로그래밍 과목을 듣는다.
> * 24명은 이산수학과 윈도우 프로그래밍을 듣는다.
> 
> 이산수학 혹은 윈도우 프로그래밍을 듣는 학생들은 시험공부때문에 파티에 참가 할 수 없다고 할 때, 파티에 참가할 수 있는 학생들의 머릿수를 구하시오.
>
> ***
> 집합 A와 집합 B를 다음과 같이 정의하기로 한다.
> * **A** = `{x|x는 이산수학 과목을 듣는 학생}`
> * **B** = `{x|x는 윈도우 프로그래밍 과목을 듣는 학생}`
> 
> 이 때 이산수학 과목 혹은 윈도우 프로그래밍 과목을 듣는 학생은 다음과 같이 구할 수 있다.
> 
>     |A ∪ B|
>       = |A| + |B| - |A ∩ B|
>       = 50 + 140 - 24 
>
>     ∵ |A ∪ B| = 166
> 
> 따라서 `U - |A ∪ B|`에 해당하는 파티에 참가할 수 있는 학생수는,
> **<center> `200 - 166 = 34` </center>** 

> 예제) 이전 예제의 학교에서 다음과 같은 추가 가정이 주어진다.
> 
> * 60명은 학부생이다.
> * 학부생중 20명은 이산수학을 듣는다.
> * 학부생중 45명은 윈도우 프로그래밍 과목을 듣는다.
> * 학부생중 16명은 두 과목을 동시에 듣는다.
> 이 때 대학원생 중 파티에 참가할 수 있는 학생 수를 구하여라.
> 
> ***
> 이전 가정에 추가 집합을 정의한다.
> * **C** = `{x|x는 학부생}`
> 
> 이 때 학부생이거나 이산수학과목을 듣거나 윈도우 프로그래밍 과목을 듣는 학생은 다음과 같다.
> 
>     |A ∪ B ∪ C|
>       = |A ∪ B| + |C| - |(A ∪ B) ∩ C|
>       = |A ∪ B| + |C| - |(A ∩ C) ∪ (B ∩ C)|
>       = |A ∪ B| + |C| - ( |A ∩ C| + |B ∩ C| - |A ∩ B ∩ C| ) 
>       = 166 + 60 - ( 20 + 45 - 16 )
>     
>     ∵ |A ∪ B ∪ C| = 177
> 
> 그러므로 대학생원중 파티에 참석할 수 있는 학생 수는,
> **<center>200 - 177 = 23</center>**

## 함수
함수는 집합 A로부터 집합 B로의 이진관계 R은, 만약 집합 A의 모든 원소 a에 대하여 `(a,b) ∈ R`인 집합 B의 유일한 원소 b가 존재하면 함수라고 불린다. 이때 A에서 B로의 함수 R에 대하여 `(a,b) ∈ R`이라는 식 대신에 `R(a) = b`라는 식을 쓸 수 있다. 이때 b는 a의 image라고 불리며 집합 A를 함수 R의 **정의구역(domain)** , 집합 B를 함수 R의 치역 또는 **공번역(range)** 라고 한다.

> 예제) 집합 A와 집합 B를 다음과 같이 정의한다.
> * **A** = `{x|x는 집}`
> * **B** = `{x|x는 색}`
>
> 여기서 A에서 B로의 함수는 무슨 집합인지 서술하라.
> ***
> A에서 B로의 함수, 즉 **`R(a) = b`** 는 집을 페인트칠 하는데 있어서 색상을 정하는것이라 할 수 있다.

함수는 그림을 표현할 수 있다. 마크다운의 한계로 그림으로 표현하기는 어려우므로 테이블로 나타낸 결과는 아래와 같다.
* `P = {a,b,c,d,e}`
* `Q = {A,B,C,D}`
*   |P  |R(p) |
    |:--|:----|
    |a  |A    |
    |b  |C    |
    |c  |C    |
    |d  |B    | 
    |e  |B    | 

### 함수의 여러가지 종류
집합 A에서 집합 B로의 전사 함수, 일대일 함수, 일대일 전사 함수는 다음과 같이 정의된다.
* 전사 함수: **B의 모든 원소 b가 a의 이미지인 경우**
* 일대일 함수: **모든 A의 원소 a가 중복되는 이미지를 가지지 않는 경우**
* 일대일 전사 함수: **전사 함수이면서 동시에 일대일 함수인 경우**

> 예제) 집합 A를 노동자의 집합이라고 하고, 집합 B, C, D를 직장의 집합이라고 하자. A에서 B는 전사함수로 정의되려 하고, A에서 C는 일대일함수로 정의되려한다. 또한 A에서 D로는 일대일 전사 함수로 정의된다고 할때 노동자의 집합이 직장에 대해서 가져야 할 관계에 대해 서술해라.
> ***
> * 그룹 B의 모든 직장에 대해 취업되지 않은 노동자가 없어야 한다.
> * 그룹 C의 모든 직장에 대해 한명 이하의 노동자가 취업되야 한다.
> * 그룹 D의 모든 직장에 대해 한명의 노동자가 취업되어야 한다.

### 역함수
f를 집합 A에서 집합 b로 일대일 전사함수라고 할 때, f의 역함수는 B에 속하는 원소 b에 대해서 `f(a) = b`인 유일한 A의 한 원소 a로 대응시키는 함수를 의미한다. 기호로는 f<sup>-1</sup>라고 나타낸다. 즉 `f(a) = b`이면 `f`<sup>`-1`</sup>`(b) = a`가 된다.

여기서 주의할 점은 함수 f의 역함수가 존재하려면 f가 일대일 전사함수여야 한다는 사실이다. 함수 f가 일대일 전사함수가 아니면 f(b)는 성립할 수 없기 때문에 함수 f의 역함수 f<sup>-1</sup>는 존재할 수 없다.

>예제) 함수 f가 정수에서 정수로의 함수로 `f(x) = x + 2`로 정의된다고 하자. 함수 f의 역함수가 존재하는가? 만일 존재한다면 역함수는 어떻게 되는가?
>***
>모든 정수 x,y에 대해서 `x ≠ y` 이면 `x + 2 ≠ y + 2`이므로 함수 f는 일대일 함수이고, 모든 y에 대하여 `x + 2 = y`를 만족하는 x가 존재하므로 함수 f는 전사함수이다. 따라서 함수 f는 일대일 전사함수이므로 역함수가 존재한다. `f`<sup>`-1`</sup>`(y) = y - 2`가 역함수가 된다.

> 예제) 함수 f가 정수에서 정수로의 함수로 `f(x) = x²`가 정의된다고 한다. 함수 f의 역함수가 존재하는가?
> ***
> f(x)에 대한 x의 값은 하나, 혹은 동시에 두개 존재하는 경우가 있으므로 함수 f는 일대일 함수가 아니다. 일대일 전사함수 또안 아니므로 역함수가 존재하지 않는다.

### 복합 함수
함수 f가 A에서 B로의 함수라고 하고 함수 g가 B에서 C로의 함수라고 하자. A에서 C로의 복합 함수 h는 모든 원소 `a ∈ A`에 대해서 `h(a) = g(f(a))`를 만족하는 복합 함수이다. `h = g ∘ f`라고 표현된다.

> 예제) 집합이 아래와 같이 정의된다.
> * `A = {a₁,a₂,a₃}`
> * `B = {b₁,b₂,b₃}`
> * `C = {c₁,c₂,c₃}`
> 또한 A에서 B로의 함수 f와 B에서 C로의 함수 g는 아래와 같이 정의된다.
> * `f(a₁) = b₁, f(a₂) = b₃, f(a₃) = b₂`
> * `f(b₁) = c₁, f(b₂) = c₃, f(b₃) = c₂`
> 이 때, f ∘ g인 h를 모든 A를 원소에 대응시켜라.
> ***
> * **<center>`f(a₁) = c₁, f(a₂) = c₂, f(a₃) = c₃``</center>** 

함수 개념과 연관하여 잘 알려진 수학적 증명 방법중 하나가 **비둘기 증명 법칙(pigeonhole)** 법칙이다. 비둘기 집 법칙은 만일 많은 비둘기가 있고 적은 수의 비둘기 집이 있으면 반드시 어떤 비둘기 집은 둘 이상의 비둘기가 들어가야만 한다는 법칙이다. 수학적으로 표현하면 다음과 같다.

D와 R을 유한 집합이라고 할 때, 만일 `|D| > |R|`이면 D에서 R로의 모든 함수 f에 대하여 `d₁, d₂ ∈ D`이며 `f(d₁) = f(d₂)`인 `d₁, d₂`가 반드시 존재한다. 이를 비둘기 집 법칙이라고 부른다.

예를 들어 만일 13명의 사람이 있다고 하면 그 둘 중 적어도 두명 이상은 같은 달에 태어났을 것이다. 이 경우 13명의 사람은 비둘기에 해당하고, 일년 12달은 비둘기 집에 해당한다.

> 예제) 한 바둑 선수가 바둑 대회에 참가하기 위해 77일간 연습 대국을 치루려고 한다. 그는 최소한 하루에 한 경기는 치루고 전체 대국 수는 132 게임을 넘지 않는다. 즉 그가 계획하고 있는 총 연습 대국 수는 77게임에서 132게임 사이이다. 그가 어떻게 스케쥴을 잡던지 특정 연속한 기간에 21게임을 하게 되는데, 이 연속한 날짜가 존재하게 됨을 증명하여라.
> ***
> a<sub>i</sub>를 i번째 날까지 치루는 총 연습 대국 수라고 하자. 그러면 `a₁, a₂, a₃, … a₇₇`은 점차 증가하는 한 순열이다. 이때 `a₁ ≥ 1`이고 `a₇₇ ≤ 132`이다. 마찬가지로 순열 `a₁ + 21, a₂ + 21, a₃ + 21, …, a₇₇ + 21`을 생각하면 이 순열도 점차 증가하는 순열이다. 그런데 순열 `a₁, a₂, a₃, …, a₇₇과 a₁ + 21, a₂ + 21, a₃ + 21, …, a₇₇ + 21`은 그 순열을 이루는 각 원소의 갯수가 154이고 가장 큰 `a₇₇ + 21은 153`보다 작거나 같다. 따라서 pigeonhole 규칙에 의거해 a<sub>i</sub>  = a<sub>j</sub> + 21인 a<sub>i</sub>, a<sub>j</sub>가 적어도 하나 존재하여야 한다. 이 말은 특정 연속 기간에 정확히 21게임을 하게 되는 연속한 날짜가 존재한다는 뜻이다.

## 재귀(recursive) 정의
재귀를 이용하여 그 정의구역이 음수가 아닌 함수를 다음과 같이 두 단계로 정의할 수 있다.

* 기반 단계 : 0일때의 함수 값을 명시
* 재귀 단계 : 특정 상수에 대한 함수 값을 더 작은 상수에 대한 함수 값으로부터 얻는 규칙을 명시

> 예제) 다음과 같이 재귀적으로 함수가 정의되어 있다고 한다.
> 
>     f(0) = 3
>     f(n + 1) = 2f(n) + 3
> 
> 이때 f(1), f(2), f(3), f(4)의 값을 구하시오.
> 
> ***
> 
> * `f(1) = 2 * 3 + 3 = 9`
> * `f(2) = 2 * 9 + 3 = 24`
> * `f(3) = 2 * 24 + 3 = 51`
> * `f(4) = 2 * 51 + 3 = 105`

> 예제) 계승 함수(factorial) F(n) = n!에 대한 재귀 정의를 보이시오
> *** 
>     F(n) = 1 (n = 0)
>     F(n + 1) = F(n)(n + 1)   (n ≥ 1)

> 예제) 피보나치 수 f<sub>n</sub>은 다음과 같이 정의된다.
> 
> * `f₀, f₁ = 1`
> * `f`<sub>`n`</sub> = `f`<sub>`n - 1`</sub> `+` <sub>`n - 2`</sub>
> 
> 이 때, `f₂, f₃, f₄, f₅, f₆`을 구하시오. 
> ***
> * `f₂ = f₀ + f₁ = 2`
> * `f₃ = f₁ + f₂ = 3`
> * `f₄ = f₂ + f₃ = 5`
> * `f₅ = f₃ + f₄ = 8`
> * `f₆ = f₄ + f₅ = 13`

## 재귀(recursive) 알고리즘
크기가 크고 복잡한 문제를 풀 때 그 문제의 필요없는 부분을 잘라내고 간소화 함으로써 주어진 문제를 풀 수가 있다. 이 절에서는 이러한 재귀 알고리즘을 다룬다.

> 예제) 양수 n에 대하여 n!을 구하는 알고리즘을 구하여라.
> ***
>     procedure factorial( n : positive int )
>     if n = 1 then
>         factorial(n) := 1
>     else
>         factorial(n) := n * factorial(n - 1)

> 예제) 피보나치(fibonacci) 수를 구하는 재귀 알고리즘을 구하여라.
> ***
>     procedure factorial( n : nonnegative integer)
>     if n = 0 || n = 1 then
>         factorial(n) := 1
>     else
>         factorial(n) := factorial(n - 1) * factorial(n - 2)

1. 다음을 증명하여라.

        R ∪ (P ∩ Q) = (R ∪ P) ∩ (R ∪ Q)

>    정답과 해설)
>
>    * `R ∪ (P ∩ Q) ⊆ (R ∪ P) ∩ (R ∪ Q)`를 증명할 때, `x ∈ R ∪ (P ∩ Q)`라면 합집합의 정의에 의하여 `x ∈ R`이거나 `x ∈ (P ∩ Q)`이어야 한다.
>        * `x ∈ R`인 경우, `x ∈ (R ∪ P) ∈ (R ∪ Q)`
>        * `x ∈ P ∩ Q`인 경우, `(P ∩ Q) ∈ (R ∪ P) ∩ (R ∪ Q)`이므로 `x ∈ (R ∪ P) ∈ (R ∪ Q)`
>    
>        위의 식에 의하여 `R ∪ (P ∩ Q) ⊆ (R ∪ P) ∩ (R ∪ Q)`이 성립한다.
>    * `(R U P) ∩ (R ∪ Q)`을 증명할 때, `R ∪ P와 R ∪ Q`의 공통적인 집합으로 R이 포함되어 있으므로 교집합의 정의에 의하여 `R ∈ (R U P) ∩ (R ∪ Q)`이다. 그러므로 R을 밖으로 분리시켜 `R ∩ (P ∪ Q)`과 같이 표현이 가능하다.
>
>        위의 식에 의하여 `x ∈ R ∪ (P ∩ Q)`이 성립한다.
>
>    첫번째 증명과 두번째 증명을 종합하면 `R ∪ (P ∩ Q) = (R ∪ P) ∩ (R ∪ Q)`이 성립한다.
